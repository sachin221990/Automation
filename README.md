# TVUniversity Coding Assignment

A coding assignment for web automation

## Setup

#### Build Project:
From terminal run
```$xslt
./gradlew build
```
#### Run tests: 

##### UI:

```$xslt
./gradlew runTests
```

##### UI on firefox:

```$xslt
./gradlew runTests -Dbrowser=firefox
```

##### Headless:

```$xslt
./gradlew runTests -Dtarget=headless
```
